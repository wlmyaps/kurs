📘 Panduan Lengkap Graph.html

Panduan interaktif untuk menguasai semua fitur dashboard kurs & saham BCA — dari command rahasia hingga analisis multi-mata uang.

🏠 Selamat Datang

Graph.html adalah dashboard HTML mandiri (standalone) untuk memantau pergerakan kurs valas BCA dan harga saham BMRI & BBCA. Semua data di-hardcode langsung di file, sehingga tidak perlu backend atau internet untuk melihat historis.

File ini berjalan 100% di browser kamu. Cukup buka file graph.html di Chrome/Edge/Firefox.

🔐 Cara Membuka Secret Menu

Secret Menu adalah pintu masuk ke fitur rahasia: update data manual, download HTML baru, dan mode Non-BCA.

🫵 Metode 1: Tap Layar 7x Cepat

Lakukan 7 kali klik/tap di area kosong mana saja pada halaman dalam jeda kurang dari 1,5 detik antar-tap. Akan muncul notifikasi "🔒 SECRET MODE ACTIVE!" dan kotak command muncul di bagian atas tengah layar.

🫵 Metode 2: Klik Tombol Close (jika sudah pernah aktif)

Jika kotak command sudah pernah muncul dan kamu menutupnya dengan tombol X CLOSE, kamu bisa memunculkannya kembali dengan cara yang sama: tap layar 7x cepat.

Jangan tap pada tombol X CLOSE atau kotak input command itu sendiri — karena tap di situ tidak akan dihitung oleh sistem secret counter.

⌨️ 4 Command Rahasia

Ketik salah satu perintah berikut di kotak command (tanpa tanda kutip), lalu tekan Enter.

#1 CSV — Update Data Manual

KETIK CSV

Perintah ini membuka form input data di panel samping kanan. Gunakan untuk menambahkan data kurs & saham terbaru secara manual.

1
Ketik CSV di kotak command → tekan Enter.
2
Panel kanan terbuka. Isi Tanggal & Jam (format: DD/MM HH:MM, contoh: 21/05 14:30).
3
Masukkan harga Buy & Sell untuk USD, SGD, AUD, JPY, EUR, CHF.
4
(Opsional) Masukkan harga saham BMRI & BBCA Open/Close jika hari kerja (Sen–Jumat).
5
Klik tombol 💾 SIMPAN DATA atau 🔄 SIMPAN & REFRESH.
Gunakan "SIMPAN & REFRESH" agar data langsung diproses dan grafik otomatis diperbarui tanpa perlu ketik REFRESH lagi.
⚠️ PENTING: Jangan pernah mengedit langsung array data di dalam kode HTML (bagian globalDates, globalCurrencyData, dll.) karena risiko corrupt encoding. Selalu gunakan form CSV ini untuk menambah data baru.

#2 REFRESH — Render Ulang Semua Grafik

KETIK REFRESH

Setelah menyimpan data via CSV, ketik perintah ini untuk memperbarui semua tabel dan grafik dengan data yang baru saja diinput.

REFRESH akan memanggil fungsi rebuildAllTablesAndCharts() yang merender ulang: tabel utama, tabel spread, grafik individu, grafik perbandingan, statistik, correlation matrix, dan alert simulator.

#3 HTML — Download File HTML Baru

KETIK HTML

Perintah ini akan mengunduh file HTML baru yang sudah berisi data terbaru (termasuk yang baru kamu input via CSV). File hasil download bisa langsung dibuka atau di-host di GitHub Pages.

1
Pastikan data sudah diinput dan di-refresh.
2
Ketik HTML → Enter.
3
Browser akan otomatis mendownload file bernama graph-YYYY-MM-DD-HHMMSS.html.
Gunakan ini sebagai backup harian. File yang didownload adalah snapshot lengkap dari state saat ini, termasuk semua data historis.

#4 NONBCA — Lihat Kurs 12 Mata Uang (Real-time API)

KETIK NONBCA

Mode ini akan membersihkan layar dan memuat dashboard kurs alternatif yang menampilkan 12 mata uang vs IDR menggunakan data real-time dari API Frankfurter.

12 Mata Uang yang Tersedia:

USDSGD JPYAUD CHFEUR HKDCNY GBPKRW MYRNZD
1
Ketik NONBCA → Enter.
2
Layar akan berubah menjadi dashboard baru dengan tampilan seperti Google Finance.
3
Pilih rentang tanggal (Mulai & Akhir) → klik Terapkan.
4
Pilih mode grafik: Nominal (IDR) atau Perubahan % (Indexed = 100 awal).
Mode "Perubahan %" sangat cocok untuk melihat mata uang mana yang paling melemah/menguat relatif terhadap IDR dalam periode tertentu.
Mode NONBCA memerlukan koneksi internet karena mengambil data dari API Frankfurter. Jika offline, tampilan akan kosong.

⏰ Cara Set Alarm

1
Isi jam (HH) dan menit (MM) di kolom Alarm.
2
Klik Set. Status berubah menjadi "✓ AKTIF".
3
Ketika waktu tiba, overlay alarm merah besar akan muncul dengan suara beep.
4
Klik 🔕 STOP untuk menghentikan alarm.
Alarm berbasis waktu Jakarta (WIB). Pastikan browser tidak di-mute agar suara terdengar.

⏱️ Countdown Timer

Masukkan jumlah detik di kolom Countdown → klik Mulai. Berguna untuk timer trading atau reminder.

🌤️ Weather (Cuaca)

Klik tombol 📍 Lokasi untuk mendeteksi lokasi kamu via browser dan menampilkan suhu serta kondisi cuaca real-time dari Open-Meteo API.

💱 Tabel Kurs Utama

Tabel pertama menampilkan seluruh data historis 6 mata uang (USD, SGD, AUD, JPY, EUR, CHF) dengan kolom Buy & Sell. Data diurutkan dari awal periode (19 April 2025) hingga terbaru.

Scroll horizontal jika tabel terlalu lebar di layar mobile. Tabel ini otomatis diperbarui saat kamu menambah data via CSV.

📊 Tabel Spread & Filter Tanggal

Section ini menampilkan selisih harga jual dan beli (Spread) untuk analisis likuiditas dan margin BCA.

1
Pilih Dari Tanggal dan s/d Tanggal di dropdown.
2
Tabel otomatis memfilter hanya data di rentang tersebut.
3
Klik 🔄 Reset 30 Hari untuk kembali ke default 30 hari terakhir.
4
Klik 📥 Export CSV untuk mengunduh data spread dalam format CSV.
Spread tinggi (warna merah pada kolom) menandakan volatilitas besar atau likuiditas rendah pada mata uang tersebut.

📉 Grafik Individual Mata Uang

Terdapat 6 grafik terpisah untuk masing-masing mata uang. Masing-masing menampilkan garis Buy (Hijau) dan Sell (Merah).

  • USD/IDR — Biru tua
  • SGD/IDR — Merah
  • AUD/IDR — Kuning/oranye
  • JPY/IDR — Hijau
  • EUR/IDR — Ungu
  • CHF/IDR — Oranye tua

📊 Google Finance Style Compare — Kurs

Fitur ini menampilkan perubahan persentase (%) dari titik awal (base 0%), persis seperti tampilan Google Finance. Sangat cocok untuk melihat mata uang mana yang paling melemah/menguat relatif terhadap Rupiah.

Mode Beli vs Jual

Gunakan tombol toggle Beli / Jual untuk beralih antara harga beli atau harga jual sebagai dasar perhitungan.

Filter Rentang (Range)

Ada dua versi grafik Google Finance Style:

  • Full Period — Menggunakan seluruh data historis yang tersedia.
  • Filter Range — Menggunakan rentang tanggal yang kamu pilih di filter Spread Table.
Jika garis naik ke atas, artinya IDR melemah terhadap mata uang tersebut (butuh lebih banyak Rupiah untuk membeli 1 unit valas).

🔍 Cara Membandingkan Hanya 2 Mata Uang (Misal: AUD vs JPY)

Secara default semua 6 mata uang ditampilkan. Untuk fokus pada perbandingan spesifik:

1
Lihat grafik Google Finance Style Compare — Kurs (Filter Range) atau (Full).
2
Klik legend (nama mata uang) di atas grafik untuk hide/show garis.
3
Klik sekali pada legend USD, SGD, EUR, CHF untuk menyembunyikannya.
4
Biarkan hanya AUD dan JPY yang aktif. Sekarang grafik hanya menampilkan perbandingan kedua mata uang tersebut!

📊 Normalized Multi-Mata Uang

Grafik Perbandingan Multi-Mata Uang — Harga Beli (Normalized) dan Harga Jual menggunakan base 100. Ini memudahkan perbandingan karena semua mata uang dimulai dari titik yang sama meski nilai absolutnya berbeda (misal: USD 16.000 vs JPY 100).

📈 Saham BMRI & BBCA

Dashboard juga memantau 2 saham blue-chip:

BMRI (Bank Mandiri)

Grafik Close (solid biru) dan Open (dash biru). Data hanya tersedia hari kerja (Sen–Jumat).

BBCA (BCA)

Grafik Close (solid oranye) dan Open (dash oranye). Data hanya tersedia hari kerja (Sen–Jumat).

📊 Filter Tabel & Grafik Saham

Sama seperti filter kurs, kamu bisa memilih rentang tanggal khusus untuk saham:

1
Pilih Dari Tanggal & s/d Tanggal di section Saham.
2
Baris tabel berwarna hijau = Close > Open (naik), merah = Close < Open (turun).
3
Grafik otomatis menyesuaikan dengan rentang yang dipilih.
Tombol 📥 Export CSV juga tersedia untuk data saham.

📈 Google Finance Style — Saham

Lihat perubahan persentase saham BMRI vs BBCA dari titik awal periode. Toggle antara mode Close dan Open.

🏆 Best & Worst Performer

Ranking otomatis yang menunjukkan mata uang/saham dengan performa terbaik dan terburuk sejak tanggal base yang kamu pilih.

1
Pilih Base Tanggal di dropdown.
2
Klik Update Ranking.
3
Card hijau (#1) = Winner. Card merah (terakhir) = Loser.

📊 Volatility Gauge

Grafik spread (selisih Sell-Buy) semua mata uang dalam satu canvas. Spread lebar = volatilitas tinggi / risiko likuiditas.

Statistik di bawah grafik menampilkan rata-rata, tertinggi, dan terendah spread per mata uang.

🔔 Price Alert Simulator

Masukkan target harga untuk memunculkan garis merah horizontal di grafik sebagai visual marker.

1
Isi target di kolom: USD, SGD, AUD, BMRI, BBCA.
2
Klik Apply Alerts.
3
Garis merah putus-putus akan muncul di grafik Alert.
4
Klik Clear untuk menghapus semua garis alert.

🔍 Dynamic Base & Zoom

Grafik interaktif dengan fitur zoom scroll dan drag pan:

  • Scroll mouse → Zoom in/out pada sumbu X (waktu).
  • Drag klik-kiri → Geser (pan) ke kiri/kanan.
  • Reset Zoom → Klik tombol reset untuk kembali ke tampilan awal.
  • Export Grafik → Download gambar PNG dari grafik yang sedang ditampilkan.
Tooltip menampilkan harga asli + persentase perubahan relatif terhadap base tanggal yang kamu pilih.

🌙 Dark Mode

Klik tombol 🌙 Dark Mode di pojok kanan atas untuk beralih ke tema gelap. Klik lagi untuk kembali ke Light Mode.

🎯 Tips Produktivitas

Rutin backup: Setiap kali selesai update data harian, ketik command HTML untuk download file terbaru sebagai cadangan.
Update sore hari: Data kurs BCA biasanya stabil setelah pukul 14:00 WIB. Input data di waktu tersebut untuk konsistensi.
Gunakan NONBCA untuk cross-check: Bandingkan data manual BCA kamu dengan data API Frankfurter untuk memastikan tidak ada typo saat input.
Perbandingan AUD vs JPY: Gunakan grafik Google Finance Style, lalu klik legend mata uang lain untuk hide. Ini lebih bersih daripada melihat 6 garis sekaligus.

🛠️ Troubleshooting

MasalahSolusi
Grafik tidak muncul setelah CSVPastikan tekan SIMPAN & REFRESH, atau ketik REFRESH di command box.
Secret mode tidak aktif setelah 7 tapPastikan jeda antar tap kurang dari 1,5 detik. Hindari mengeklik tombol atau link saat men-tap.
Data saham tidak tersedia akhir pekanNormal. Pasar saham tutup Sabtu-Minggu. Biarkan kolom saham kosong saat input CSV di hari libur.
NONBCA blank / tidak loadPastikan koneksi internet aktif. Mode NONBCA membutuhkan fetch API ke frankfurter.dev.
File HTML hasil download corruptJangan edit file hasil download dengan text editor biasa yang bisa merusak encoding. Langsung buka di browser.
Alarm tidak bersuaraBrowser mungkin memblokir audio autoplay. Klik mana saja di halaman sekali untuk mengaktifkan AudioContext sebelum alarm tiba.
